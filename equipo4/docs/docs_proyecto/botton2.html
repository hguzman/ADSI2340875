7.1. Incidentes

  • Ataque por virus.
  • El aplicativo no muestra correctamente el precio de los productos.
  • Varios usuarios no pueden registrarse de manera correcta para hacer pedidos, por fallas en el aplicativo.
  • El cliente debe suministrar más información acerca de la directriz o del impacto de una acción correctiva.

7.2. Tiempo de respuesta

  • Si el nivel de prioridad que requiere la directriz es máximo, tendrá un tiempo de respuesta inmediato.
  • Si el nivel de prioridad que requiere la directriz es alto, tendrá un tiempo de estimado de (4 hora).
  • Si el nivel de prioridad que requiere la directriz es medio, tendrá un tiempo de estimado de (8 horas).
  • Si el nivel de prioridad que requiere la directriz es bajo, se estaría a la espera de respuesta del cliente.

7.3. Tiempo de solución

  • El tiempo de solución estimado para las directrices de carácter inmediato será de (6 horas).
  • El tiempo de solución estimado para las directrices de carácter alto será de (12 horas).
  • El tiempo de solución estimado para las directrices de carácter medio será de (2 días).
  • El tiempo de solución estimado para las directrices de carácter bajo será de (3 a 4 días).
